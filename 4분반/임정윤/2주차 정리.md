1. 인공지능의 계층 구조
 AI (Artificial Intelligence): 인간의 지능을 기계로 모사하는 포괄적인 기술
 ML (Machine Learning): 데이터를 통해 기계가 스스로 규칙을 학습하는 AI의 한 분야
 DL (Deep Learning): 인공신경망을 활용해 복잡한 패턴을 학습하는 ML의 하부 기술
 관계: AI ⊃ ML ⊃ DL (AI가 가장 크고 그 안에 ML, 그 안에 DL이 포함됨)

2. 학습 방식에 따른 분류
 지도학습 (Supervised Learning): 문제(Feature)와 정답(Label)을 함께 학습시키는 방식. 회귀(Regression)와 분류(Classification)가 대표적
 비지도학습 (Unsupervised Learning): 정답 없이 데이터의 특성만으로 구조를 파악하는 방식. 군집화(Clustering)가 대표적

3. 데이터의 형태
 수치형 데이터 (Numerical Data): 숫자로 표현되며 연산이 가능한 데이터 (예: 키, 몸무게, 가격)
 범주형 데이터 (Categorical Data): 범주나 종류를 나타내는 데이터 (예: 성별, 혈액형, 지역). 컴퓨터가 이해하려면 숫자로 변환하는 과정이 필수

4. 데이터 정제 (Cleansing)
 결측치 (Missing Value): 수집되지 않은 빈 값 ($NaN, Null$). 평균값/중앙값 대체 또는 해당 행 삭제로 처리
 이상치 (Outlier): 일반적인 데이터 분포에서 크게 벗어난 값. 모델의 왜곡을 방지하기 위해 제거하거나 조정 필요

5. 데이터 스케일링 (Scaling)
 특성마다 값의 범위가 다를 경우 모델이 특정 특성에만 편향될 수 있어 단위를 맞추는 작업임
  표준화 (Standardization): 평균을 0, 표준편차를 1로 변환. 데이터가 정규분포를 따를 때 유리함
   공식: $z = \frac{x - \mu}{\sigma}$
  정규화 (Normalization): 모든 값을 0~1 사이로 압축. 데이터의 최소/최대값을 기준으로 함
   공식: $x_{new} = \frac{x - x_{min}}{x_{max} - x_{min}}$

from sklearn.preprocessing import StandardScaler, MinMaxScaler

# 표준화 적용
std_scaler = StandardScaler()
df_std = std_scaler.fit_transform(df)

# 정규화 적용
mm_scaler = MinMaxScaler()
df_mm = mm_scaler.fit_transform(df)

6. 범주형 변수의 수치화 (Encoding)
 글자로 된 범주형 데이터를 숫자로 변환하는 기법
  원-핫 인코딩 (One-hot Encoding): 각 카테고리를 독립된 차원으로 만들고 해당하는 위치에만 1, 나머지는 0을 부여하는 방식
   장점: 범주 간 순서가 없을 때 적합
   단점: 카테고리가 많아지면 차원이 너무 커짐
  해싱 (Hashing): 해시 함수를 통해 데이터를 고정된 크기의 숫자로 변환
   장점: 메모리 효율이 좋고 차원 관리가 용이함

# One-hot Encoding 예시
import pandas as pd
df_encoded = pd.get_dummies(df['category_column'])
